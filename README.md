## Sorting Hat
This Assignment sorts students, assigns houses randomly and allows for expulsion of students that are no longer attending. 

## Getting Started
```
$ git clone git@github.com:Lakiajo/Sorting-Hat.git
```

### Prerequisites
```
$ npm install -g http-server
```
